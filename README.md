# Random-SVG-image-generator
Creates a random SVG image

C

`
cc rand_image.c -Wall -o rand_image && ./rand_image
`

or

`
clang rand_image.c -Weverything -o rand_image && ./rand_image
`

Python

`
python rand_image.py 
`

or

`

python3 rand_image.py

`
